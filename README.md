# What today's holiday?

## This app shows today's holiday.


## How to build:

### Arch Linux install:

Auto Install Kivy:

`./scriptKivyProjectInstall.sh `

### Building:
```
buildozer android debug
  
Apk file -> ./bin/*.apk
Start app -> python main.py
```
